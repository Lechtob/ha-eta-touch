# Security Policy

## Supported versions

Security fixes are provided for the latest released version of ETA Touch for Home Assistant.
Please reproduce an issue with the latest release before reporting it when possible.

## Reporting a vulnerability

Do not report security vulnerabilities in a public issue. Use GitHub's private
[security advisory form](https://github.com/Lechtob/ha-eta-touch/security/advisories/new).

Include the affected integration version, Home Assistant version, ETA model, impact, and
minimal reproduction details. Remove credentials, external Home Assistant URLs, serial
numbers, and unrelated device data.

You should receive an initial response within seven days. Details will remain private until
a fix or coordinated disclosure is ready.
